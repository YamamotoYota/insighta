# INSIGHTA 配布版（実行方法）

## 1. このフォルダにあるもの
- `INSIGHTA.exe` : アプリ本体
- `README.md` : この説明書

## 2. 起動方法
1. `INSIGHTA.exe` をダブルクリックします。
2. 数秒待つと、既定のブラウザで `http://127.0.0.1:8050` が自動で開きます。
3. 画面が表示されたら利用開始できます。

補足:
- 自動でブラウザが開かない場合は、手動で `http://127.0.0.1:8050` を開いてください。
- 初回起動時は展開処理のため少し時間がかかる場合があります。

## 3. 終了方法
- アプリ画面上部の `アプリを終了` ボタンを押してください。
- 終了ボタンを押すと、ローカルサーバーを停止します。
- 停止後はブラウザタブを閉じてください。

## 4. 動作環境（配布版）
- Windows 64bit
- Webブラウザ（Edge / Chrome など）

この配布版には Python 実行環境と主要ライブラリを同梱しています。
通常、Python や仮想環境の準備は不要です。

## 5. データ入力
- CSV
- Excel（`.xlsx`, `.xls`, `.xlsm`）
- SQLデータベース接続（DBMSに応じて追加ドライバが必要な場合あり）

## 6. SQL接続を使う場合の注意
DBMSごとに、OS側または追加ライブラリ/ドライバの準備が必要な場合があります。

- SQL Server: ODBC Driver 17/18 for SQL Server
- MySQL
- SQLite
- Oracle Database
- PostgreSQL

※ SQL Server は特に ODBC ドライバ未導入だと接続できません。

## 7. よくあるトラブル

### 7.1 画面が開かない
- `INSIGHTA.exe` 起動後、10〜30秒ほど待ってください（初回起動は遅いことがあります）。
- ブラウザで `http://127.0.0.1:8050` を直接開いてください。

### 7.2 `8050` ポートが使われている
- すでに別の INSIGHTA や他アプリが起動している可能性があります。
- 既存のアプリを終了してから再度起動してください。

### 7.3 Excel ファイルが読めない
- 配布版では Excel 読み込み用ライブラリを同梱しています。
- それでも読めない場合は、ファイルが破損していないか、拡張子と実体が一致しているかを確認してください。

## 8. セキュリティに関する補足
- 本アプリはローカルPC上で `127.0.0.1`（自分のPC内）にWeb画面を表示します。
- 外部公開サーバーではありません。
